The instructions to run the code 
-------------------------------------------------------------------------------------------------------------------------------------------
1. The major requirement is Python 3.6.1. This must be installed on the system with the numpy and tensor flow modules.
2. After downloading the file, place the file in any folder.
3. Open the Command Prompt in Windows (or) Terminal in Mac.
4. Go to the directory in which the file is placed.
   c:\> cd directoryname
5. Now run the file using the command "python3 train_and_eval.py"
6. The output of the program at various steps is obtained one after the other. Observe the reduction in Reconstruction error of Bob and the increase in Reconstruction error of Eve. Wait until the program terminates. The target losses are achieved.
